Bienvenue sur Notflix !

La contrefaçon lowcost de Netflix.

Ce projet à pour but de vous faire concevoir et développer des interfaces web pour interagir avec des services multimedia d’UPnP (mediaServer, mediaRenderer, norme DLNA) présents sur nombre d’appareil du commerce (TV, cadre photo, lecteur audio, …) ainsi que via de nombreux programmes pour ordinateurs (ex: Plex, Kodi), tablettes et téléphones (ex : Bubble UPnP). Vos interfaces devront permettre aux utilisateurs de naviguer dans les bases de médias (musiques, images, vidéos) et de piloter la lecture de ces médias sur les lecteurs dédiés (ex : Kodi).

Un serveur NodeJS vous est fourni, il permet d’accéder aux services UPnP présents sur le réseau local. Ce serveur permet de faire bien d’autres choses mais nous nous contenterons de cet aspect pour ce projet. Vous devrez cloner le code du serveur dans un projet github qui vous soit propre afin de travailler dessus. Il vous sera possible d’étendre le serveur selon vos besoin par la suite (ex: pour gérer l’interaction entre plusieurs plateformes comme un PC et des téléphones).

La partie cliente sera à coder par vos soin. Il faudra la développer de sorte à en permettre l’utilisation sur des plateformes aux caractéristiques différentes (PC, tablette, téléphone, écran passif + téléphone, …). Le choix des technologies à utiliser est libre, vous pouvez utiliser une technologie web (par exemple Angular 1.5.x vue en L3) ou pas (par exemple java SWING, openGL, …).
